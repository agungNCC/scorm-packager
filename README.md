# Bawana SCORM Packager v1.0

Aplikasi untuk mengkonversi file PDF dan file hasil publish-an Genially menjadi paket SCORM 1.2 yang siap diunggah ke LMS.

## Fitur

- **Konversi Genially ke SCORM** — Upload file ZIP hasil publish Genially
- **Konversi PDF ke SCORM Flipbook** — Upload file PDF, tampil dengan efek flipbook interaktif

## Langkah Penggunaan

1. **Upload File** — PDF atau ZIP (drag & drop atau klik)
2. **Convert** — Klik tombol Process untuk memulai konversi, progress ditampilkan real-time
3. **Nama File SCORM** (opsional) — Isi kolom Output Name, atau biarkan dengan nama default
4. **Download** — Klik Download untuk mendapat file SCORM (.zip)
5. File SCORM siap diunggah ke LMS
